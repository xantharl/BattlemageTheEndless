1. Download the ArbutusSlab-Regular font from https://fonts.google.com/specimen/Arbutus+Slab

2. Import font: https://dev.epicgames.com/documentation/en-us/unreal-engine/importing-fonts-in-unreal-engine

3. Go to 'Content/fantasy_gui_4/fonts' and double-click the 'f_font1.uasset'. 

4. On the Composite Fonts tab, update the font face by opening the SelectBox and selecting the font you imported in step 2.

5. Save changes.
